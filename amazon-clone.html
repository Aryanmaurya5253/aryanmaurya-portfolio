Amazon Clone (Web UI)

Overview

This project replicates the Amazon shopping UI using HTML, CSS, and JavaScript. It includes product grids, category filtering, a responsive design, and a shopping cart simulation for an e-commerce interface experience.

Technologies Used

  • HTML5 & CSS3
  • JavaScript (Vanilla)
  • Responsive Web Design (Flexbox & Grid)
